# adopt-pets

## About
```
A multi-page vue app to learn and practice vue concepts like: router, actions, getters, mutations, and state
```

```
Using custom features: 
Babel, Router, Vuex, CSS Pre-processors, Lintor

linter: standard
lint on save
dedicated config files
npm i bootstrap-vue bootstrap
```
